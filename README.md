# Jewelry Website

A multi-page storefront website for a jewelry brand, built as a static frontend project with landing, shop, blog, and about pages.

## Features

- Multi-page navigation for home, shop, blog, and about content.
- Product-focused landing page with featured items and imagery.
- Brand-style layout suitable for an ecommerce presentation demo.
- Static frontend structure that can be extended into a larger store experience.

## Tech Stack

- HTML
- CSS
- JavaScript

## Getting Started

- Clone or download this repository.
- Open index.html in your browser, or run the folder with VS Code Live Server.


## Notes

- The repository name keeps the original spelling, while the README uses the corrected project title.
- Use a local server if you want smoother navigation while testing multiple pages.
